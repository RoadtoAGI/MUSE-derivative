# 对白规则

本文件整理麦基《对白》中的核心对白规则，供对白创作时参考。

---

## 对白的六项任务

根据麦基《对白》，好的对白同时承担以下任务：

### 原文引用

> 「对白比其他赖以塑造角色的技巧（性别、年龄、衣着、阶级、演员阵容）都更有力地将故事提升到多面的生命层次，把复杂讲述的简单故事发展成全面的复杂性。」
> —— 《对白》前言

### 六项任务

1. **暴露**：传达读者需要知道的信息，但必须在冲突中自然流出
2. **推进**：推动故事前进，不是原地踏步
3. **塑造**：揭示人物性格——人物说什么和怎么说暴露他是谁
4. **差异化**：区分不同角色——每个人物有独特的声音
5. **注释**：为场景增添情感色调和意义层次
6. **节奏**：控制场景的速度和呼吸

## 对白的四类缺陷

### 1. 冗余对白

说了读者已经知道的事，或重复说同一件事。

| 表现 | 修正 |
|------|------|
| "正如我们上次讨论的……" | 直接推进到新信息 |
| 多个角色说同样的意思 | 一个角色表达就够 |

### 2. 说教对白

人物变成了作者的传声筒，说出主题思想。

| 表现 | 修正 |
|------|------|
| "你知道吗，人生最重要的是……" | 让行动传达主题 |
| 旁白解释意义 | 删除，信任读者的理解力 |

### 3. 无冲突对白

两个人友好地交换信息，没有对抗。

| 表现 | 修正 |
|------|------|
| A问B答，平顺流畅 | 加入目的对立——A想知道B想隐瞒 |
| 寒暄、天气、近况 | 除非寒暄下面有潜台词 |

### 4. 声音同质

所有角色用相同的词汇、句式和节奏说话。

| 表现 | 修正 |
|------|------|
| 遮盖名字无法分辨 | 回溯 8D 声音标记（dialogue-craft SKILL.md §2 消费法） |
| 所有人都说完整优雅的句子 | 根据性格调整——有人结巴、有人粗暴、有人迂回 |

## 验证清单

- [ ] 每句台词是否有明确的言语行动（用一个动词描述）？
- [ ] 对话中是否有冲突（目的对立）？
- [ ] 是否有潜台词（已说 ≠ 真实意图）？
- [ ] 声音差异是否足够（盖住名字测试）？
- [ ] 信息暴露是否自然（在冲突中流出）？
- [ ] 是否避免了冗余、说教、无冲突、同质四类缺陷？
- [ ] 是否符合下方"对白交流契约"？

---

## 对白交流契约

> **核心原则**：对白必须**优先服务场内交流**——给场内对象传递信息或制造反应，**而不是**服务读者解释、作者抒情或制造假悬念。

> 真实的说话总要回答："谁说？谁听？想让对方得到什么信息或反应？哪些信息不能说穿？" 自言自语也要符合自我对话逻辑——是在对自己整理 / 说服 / 质问，**不是说给读者听**。

### 4 类失败形态

| 失败 | 实证 | 反向追问 |
|------|------|----------|
| **无场内理由的隐语 / 哑谜** | 单字回应（"重订" / "古墓"），无密语约定、无防偷听、无装腔铺垫 | 听者听懂了吗？为什么说话人选这种表达方式？ |
| **对已知者重述已知信息** | "正如我们上次讨论的……"、给同伴普及只有读者不懂的设定 | 听者已知，说话人为什么还说？ |
| **省略到接收者无法恢复指代** | "你要量第一念，"（句子未完 / 主语缺失） | 接收者能听懂吗？还是只有读者要靠脑补？ |
| **文学化玄言替代有效信息** | 每句双重含义、对仗排比、抒情堆叠 | 删掉文学修辞后还剩多少**信息**？ |

### 豁免（一句话）

**装腔作势 / 江湖暗号 / 防第三方偷听 / 自言自语**——可以成立，但必须有**场内动机**和**接收逻辑**支撑（不是作者偷懒的借口）。

### 修订方向

减法为主：能删则删；不能删的换成具体动作 / 物件反应 / 对方反应承载节奏；不要把哑谜改得更玄、把演讲改得更优雅——根因不变。
