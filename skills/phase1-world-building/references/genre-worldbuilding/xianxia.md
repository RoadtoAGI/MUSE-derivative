# 仙侠（xianxia）世界观参考

> 本文不复述 SKILL.md §3 的通用规则。canon-distill 当前**无典型仙侠锚点**，本文以 driver 范畴 + 平台样态 + 反例为主——待 canon 入库后回填实证锚点。

## generative_driver 选型范畴

仙侠 driver 必须回答四件事：

- **灵气来源 / 修行能量论**：天地灵气从哪来、为什么稀缺、如何吸纳、能否被垄断；这是仙侠的"物理公理层"
- **修行阶梯 / 境界体系**：从凡人到飞升的完整阶梯（炼气 / 筑基 / 金丹 / 元婴 / 化神 / 渡劫 / 飞升 等）；每一阶的代价、必备条件、突破机制
- **门派 / 宗门 / 资源垄断结构**：哪些组织控制哪些资源（灵脉 / 法器 / 丹药 / 秘籍 / 洞天福地）；江湖 vs 仙门的接口
- **法器 / 秘境 / 因果机制**：物的能力是否可被规则化（不只是"主角拿到神兵"）；秘境的进出条件 / 因果报应是否可计算

仙侠和武侠的关键差异：仙侠有**修行的形而上学**（灵气论 + 飞升轨迹 + 渡劫机制），武侠仅有**武学技术层**。仙侠 driver 必须含形而上学层。

### platform_mode vs classic_mode

| 模式 | 特征 |
|------|------|
| **platform_mode** | 境界等级数字化 / 系统流 / 修炼速率明示 / 装备升级清晰 / 副本结构 | 起点 / 番茄主流仙侠（凡人流 / 洪荒流 / 玄幻仙侠） |
| **classic_mode** | 修行有伦理重量 / 境界差异不只是力量也是价值观 / 仙人有"非人化"代价 | 暂无典型 canon-distill 锚点 |

主流仙侠基本都走 platform_mode。classic_mode 仙侠（如《诛仙》《佛本是道》等）更注重道与人的张力，但 canon 还未入库。

## 锚点补全（待补 canon）

canon-distill 当前缺典型仙侠：

- **凡人流代表**：凡人修仙传（忘语） — 修行体系清晰，代价感强
- **东方仙侠代表**：诛仙（萧鼎） — 正邪不二元，仙凡有伦理重量
- **古典仙侠**：西游记 — 神魔体系作为东方奇幻语法源
- **网文经典**：莽荒纪 / 遮天 — 玄幻仙侠杂交，境界数字化经典

## 驱动元素清单（未有 canon 锚点时的 driver 选型抓手）

仙侠 driver 至少覆盖以下要素：

- **灵气模型**：灵气如何分布（地脉 / 灵山 / 灵气复苏 / 三千世界）；是否可被垄断 / 是否会枯竭
- **修行阶梯**：明确的境界划分（炼气 / 筑基 / 金丹...）；每阶突破的必备条件（资源 / 心境 / 因果 / 雷劫）
- **修行代价**：寿元增减 / 心魔 / 走火入魔 / 因果债 / 雷劫存活率
- **门派 / 宗门 / 散修生态**：组织化修士 vs 散修；资源分配机制（任务 / 拍卖 / 秘境）
- **法器 / 丹药 / 秘境 / 阵法**：是否可被规则化运作（不只是"主角拿到")；获取机制 + 使用代价
- **天劫 / 飞升 / 仙凡接口**：渡劫机制；飞升后是否还有故事；仙界 vs 凡界的物理 / 礼制接口
- **正邪 / 善恶的运作**：是否纯二元；魔修 / 邪修是否也有自己的修行伦理；正派内部是否同样有伪善

## 平台样态参考

观察时间：2026-05 范围。

- **起点 / 番茄常见钩子**：凡人流（《凡人修仙传》一脉） / 灵气复苏（现代背景 + 修真） / 神话流（《尔等修真者》以中国神话为底） / 系统流（任务奖励灵石 / 功法） / 大佬重生（带前世记忆速通修行）
- **盐言常见钩子**：短篇仙侠言情（修真 + 情感双线） / 修罗场（多女主 / 兄弟同修） / 一卷一境的小本结构
- **东方仙侠 vs 西方奇幻**：仙侠以"修行 / 飞升"为核心叙事弧（人 → 神），西方奇幻以"任务 / 拯救"为核心（凡人 → 英雄）；仙侠 driver 的形而上学层不可省

## 高频禁区

| 禁区 | 表现 | 修正 |
|------|------|------|
| 无形而上学层 | 只写门派 / 比武 / 拜师，不写灵气论 / 修行机制 / 境界本质 | 至少给灵气来源 + 境界阶梯 + 修行代价三件套 |
| 修行无代价 | 境界突破靠"灵光一闪" / "天降机缘" 无前置 / 无后果 | 突破须有 资源 + 心境 + 因果 + 雷劫 至少一项作为代价 |
| 境界数字化但无差异 | 炼气一二三...九十九层，但层间无质变差异 | 大阶（炼气 / 筑基 / 金丹...）之间应有质变（寿元 / 神通 / 法理理解层级），小层只是渐变 |
| 把"修仙 / 仙侠"当类型即 driver | "玄幻仙侠型" / "正统东方仙侠型"——只是类型分类，不是 driver | driver 应到灵气论 + 境界阶梯 + 资源结构 具体层 |
| 飞升后无世界 | 写到化神 / 渡劫就草草飞升，仙界没有任何设定 | 若故事跨仙凡边界，仙界须有自己的物理 / 礼制 / 资源体系（不必详尽，但接口要清楚） |
| 法器 / 丹药通用化 | 任何法器拿来即用，无适配 / 无消耗 / 无副作用 | 法器有契合度 / 灵气消耗 / 反噬可能；丹药有冲突可能（同类不可叠加） |
