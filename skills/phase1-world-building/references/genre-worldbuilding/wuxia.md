# 武侠（wuxia）世界观参考

> 本文不复述 SKILL.md §3 的通用规则。武侠 driver 与历史背景 / 江湖秩序高度耦合，下面给具体取材。

## generative_driver 选型范畴

武侠 driver 必须回答四件事：

- **武学体系运作方式**：内功 / 外功 / 心法 / 招式如何运作、修炼周期、走火入魔机制、武学进阶代价
- **江湖秩序**：门派 / 帮会 / 镖局 / 武林声望系统；江湖 vs 庙堂的接口
- **武学传承与稀缺资源**：秘籍 / 师承 / 奇遇 / 内力源；什么决定了"为什么这个人比那个人强"
- **历史框架约束**：朝代 / 战乱 / 国族冲突如何渗透江湖——武侠故事的"外部限制"

driver 选定后，Phase 5/6 才知道场景里的比武 / 拜师 / 江湖恩怨如何展开。

### platform_mode vs classic_mode

| 模式 | 特征 |
|------|------|
| **platform_mode** | 武学等级清晰（境界 / 段位） / 系统流 / 主角光环明显 / 装备升级感 | 起点 / 番茄爽文向武侠（与玄幻边界模糊） |
| **classic_mode** | 武学有道德重量 / 江湖与历史同步崩塌 / 善恶非二元 | 金庸 / 古龙 / 温瑞安传统 |

## 名著锚点（few-shot）

### 《射雕英雄传》— 五绝格局 + 武学道德捆绑

driver 拆解：

- **武学体系**：内功为根基、外功为枝叶；内功需数十年积累、速成必走火入魔（欧阳锋逆练经脉）
- **江湖秩序**：五绝格局（东邪西毒南帝北丐中神通）= 五种价值观；华山论剑定天下第一
- **传承与稀缺**：《九阴真经》是顶级秘籍但"不练"才是最高境界；师承是身份和道德的根
- **历史框架**：宋金蒙三国并立；江湖与庙堂深度纠缠（丐帮抗金 / 全真教入世 / 蒙古汗权）

派生 world_rules 关键条：
- 物理：地理塑造武学流派（大漠 / 中原 / 海岛 / 西域）；速成必走火入魔
- 社会：宋金蒙博弈；武林声望即权力但终为虚名
- 心理：侠义伦理（师恩 / 国仇 / 报恩）；"邪不胜正"是心理优势不是武力对比；血统不决定人格（杨康汉人认贼 / 郭靖蒙古认宋）

参考产物：MUSE-canon-distill `knowledge-base/novels/射雕英雄传/pipeline/phase1_world.yaml`。

### 《神雕侠侣》— 武学传承 + 礼教冲突（续作）

driver 拆解（在射雕基础上）：

- **武学体系**：玉女心经 / 黯然销魂掌 / 独孤求败剑法等新武学层；杨过的"非正统传承"对照郭靖的"师门正统"
- **江湖秩序**：第二代江湖人物登场；丐帮内乱 / 古墓派封闭 / 桃花岛传承
- **新约束**：师徒之恋的礼教冲突（杨过 vs 小龙女）作为外部限制
- **历史框架**：襄阳保卫战；郭靖夫妇守城是历史结局的预兆

参考产物：MUSE-canon-distill `knowledge-base/novels/神雕侠侣/pipeline/phase1_world.yaml`。

## 武学等级体系的内部一致性

武功体系最易作弊。Phase 1 应明确：

- **进阶代价**：每升一阶需要什么（年功 / 奇遇 / 秘籍 / 内力）
- **不可跳级**：从二流到一流到顶尖须有可叙述的因果链
- **走火入魔机制**：什么操作会导致破坏（逆练经脉 / 心魔 / 内力反噬）
- **谁是天花板**：顶级强者数量稀少（金庸：五绝；古龙：兰花传奇人物）

主角武功提升必须每一步都可追溯——禁忌"一夜顿悟天下无敌"无前置铺垫。

## 平台样态参考

观察时间：2026-05 范围。

- **起点 / 番茄常见钩子**：境界流（修炼等级化数字化） / 系统武侠（任务奖励武功） / 江湖游历（单本结构清晰） / 名门正派出身 vs 邪派传人对照 / 江湖 + 朝堂双线
- **平台与古典差异**：网文武侠倾向 platform_mode 的"明等级"，金庸古龙倾向 classic_mode 的"暗等级 + 道德重量"

## 高频禁区

| 禁区 | 表现 | 修正 |
|------|------|------|
| 武功无代价 | 主角突破靠"灵光一闪" / "突然顿悟" | 任何武功提升须有可叙述的奇遇 / 苦练 / 师承 / 因果链 |
| 江湖与历史脱节 | 江湖独立于朝代 / 战乱 / 国族冲突存在 | 江湖与庙堂应有接口（抗金 / 通敌 / 入世 / 隐世）；历史事件作为外部约束 |
| 善恶二元 | 正派绝对正义 / 邪派绝对邪恶 | 即便正派核心人物（如黄药师 / 段智兴）也有道德瑕疵；邪派可有自己的伦理（古龙更甚） |
| 把"江湖"当 driver | "江湖恩怨型武侠" / "门派斗争型武侠" | 这些是后果。driver 是具体的武学体系 + 江湖秩序 + 历史框架的组合 |
| 跳级武功 | 主角短期从三流到天下第一无前置 | 等级提升必须每一步有铺垫；天才主角也需有可追溯的成长链（如郭靖的多重奇遇叠加） |
