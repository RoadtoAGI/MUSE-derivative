# 宫斗 / 权谋（palace-intrigue）世界观参考

> 本文不复述 SKILL.md §3 的通用规则。canon-distill 当前缺典型宫斗锚点，本文用白鹿原（宗族权力向）+ 大唐李白（门第制度向）作类比 + 平台样态补足。

## generative_driver 选型范畴

宫斗 / 权谋 driver 必须回答四件事：

- **权力结构**：等级秩序、继承规则、册封 / 赏罚机制、礼制周期（选秀 / 侍寝 / 册封 / 失势 / 牵连）
- **信息流动**：内廷 vs 外朝信息差；宦官 / 外戚 / 谍报的信道；谁能见到皇上 / 谁不能、谁能传话 / 谁被监视
- **资源分配**：宫殿配置 / 月俸 / 仆从数量 / 生育权 / 储位决定权——这些都是 Phase 5/6 的具体冲突资源
- **道德 / 礼制约束**：什么行为会被审判（不孕 / 越制 / 私通） / 什么行为受奖赏；礼制不是装饰，是真实的运作系统

### platform_mode vs classic_mode

| 模式 | 特征 |
|------|------|
| **platform_mode** | 权力结构清晰 / 升级路径明确（贵人→嫔→妃→皇后） / 反转密度高 | 起点女生 / 番茄宫斗（甄嬛 / 如懿系） |
| **classic_mode** | 权力运作内化为日常生活密度 / 家族 / 礼制的共同压力 | 红楼梦传统 / 严肃历史小说 |

## 名著锚点（few-shot）

### 《白鹿原》— 宗族权力 + 礼制运作（类比锚点）

driver 拆解（不是宫廷宫斗，但宗族权力结构 / 礼制运作有强可比性）：

- **权力结构**：宗法制度 → 族长拥有祠堂惩罚权（刺刷 / 跪罚） / 决定族人婚丧嫁娶；保甲制 / 联保制 → 国家权力通过中间人延伸到乡村
- **信息流动**：宗祠是权力中心；田福贤 / 岳维山等中间人是国家与村庄的接口
- **资源分配**：土地是唯一真实财富 → 换地 / 卖地 / 分地贯穿全书；婚姻即政治（冷先生做媒联姻白鹿两家）
- **道德 / 礼制约束**：乡约制度（朱子《乡约》）作为道德自治；"面子高于一切"（白嘉轩驼背也不弯腰）；女性物化（田小娥被全村道德审判）

派生 world_rules 关键条：
- 物理：黄土高原台塬地貌——封闭性使村庄自成世界；干旱与瘟疫是周期性天灾
- 社会：宗法制度 / 乡约 / 土地阶级 / 婚姻即政治
- 心理：忍耐 / 隐忍是最高美德；面子高于一切；女性被物化；鬼神信仰与儒家理性并行

**为什么作类比锚点**：宫斗本质 = "封闭权力空间内的资源 / 信息 / 礼制 / 道德博弈"，白鹿原的宗族权力空间与之同构（祠堂 ≈ 后宫；族长 ≈ 皇上；族规 ≈ 后宫制度；冷先生媒妁 ≈ 选秀指婚）。借用其权力运作密度的写法，避开"只写勾心斗角不写制度"的常见 platform_mode 缺陷。

参考产物：MUSE-canon-distill `knowledge-base/novels/白鹿原/pipeline/phase1_world.yaml`。

### 《大唐李白》— 门第制度作为外部权力约束

driver 拆解（不是宫斗，但门第制度是权谋小说的常见 driver）：

- **权力结构**：门第制度（皇室李家以姓氏 / 郡望 / 家世定义社会地位）；皇室本身也在重写身世（"陇西成纪"李广后裔）
- **信息流动**：驿递系统 + "过所"（通行证）= 帝国信息 + 身份审查体系；商人之子在驿路上的身份审查构成持续威胁
- **资源分配**：科举资格 = 身份壁垒（"选人"与"任官"已是两条路）；干谒与荐举是科举外路径
- **运作机制**：李白家族"改姓李"= 仿效皇室重写身世；李白通过诗文 + 干谒入仕

参考产物：MUSE-canon-distill `knowledge-base/novels/大唐李白/pipeline/phase1_world.yaml`。

## 驱动元素清单（未有 canon 锚点时的 driver 选型抓手）

宫斗 driver 需要至少覆盖以下要素（不是必须全有，但选 3-4 项作为本作 driver 主体）：

- **后位 / 储位继承法**：嫡长 / 立贤 / 立爱 / 立长 哪条主导？这决定整本书的权力博弈方向
- **选秀 / 册封 / 升降的周期与规则**：什么时机选秀 / 什么条件升妃 / 什么过错降位 / 什么情况赐死
- **生育政治**：皇子的数量与归属 / 抚养权 / 储位竞争；不孕 / 难产 / 子嗣早夭的政治后果
- **内廷外朝信道**：哪些人能传话 / 哪些人被禁言；宦官 / 外戚 / 太医 / 嬷嬷的信息中介角色
- **礼制日历**：节庆 / 祭祀 / 生辰 / 忌日 = 权力展示与重组的固定时机
- **起居 / 仪式 / 服饰 / 用度的等级化**：什么份位用什么样的服饰 / 仪仗 / 月俸 / 仆从 / 殿宇——这是 Phase 5/6 场景的具体冲突资源

## 平台样态参考

观察时间：2026-05 范围。

- **起点女生 / 番茄常见钩子**：宫斗（嫡长继承 / 谋皇后位 / 大女主复仇） / 宅斗（家族内嫡庶斗争） / 重生（带前世记忆复仇） / 穿越（现代女主带知识入古代） / 影后嫡女 / 大女主权谋
- **盐言常见钩子**：单元高反转（一卷一斗） / 史料 + 戏剧化融合（援引《清史稿》 / 后宫档案）

平台样态的常见亮点：援引真实史料（《清史稿》、后宫档案）作为礼制 / 制度的依据——这避免了"凭空发明礼制"。

## 高频禁区

| 禁区 | 表现 | 修正 |
|------|------|------|
| 把"勾心斗角"当 driver | "宫斗型宫斗" / "权谋型权谋" / "斗争型"——本质是后果，不是机制 | driver 应到具体的权力结构 + 继承规则 + 信道 + 礼制周期 |
| 礼制 / 制度被忽略 | 只写人物算计 / 对话 / 暗箭，不写制度运作 | 礼制不是装饰是运作系统——什么礼数 / 什么仪仗 / 什么月俸都是真实的冲突资源 |
| 信道 / 信息差悬空 | 主角"突然知道"或"突然不知道"关键信息 | 信息差应有具体信道（宦官 / 外戚 / 嬷嬷 / 谍报 / 太医） |
| 资源 / 升降无依据 | 妃位升降 / 储位归属凭"皇上喜恶"决定，无章法 | 至少应有可叙述的章法（嫡长 / 立贤 / 子嗣 / 家世），即便"皇上喜恶"也应在章法下博弈 |
| 把"权谋型"当抽象类型 | 宫斗等于"勾心斗角"，不区分内廷 / 外朝 / 家族 / 朝廷 | driver 应明确权力空间（后宫 / 朝堂 / 家族 / 派系），不同空间的运作机制差别极大 |
