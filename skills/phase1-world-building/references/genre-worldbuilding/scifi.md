# 科幻（scifi）世界观参考

> 本文不复述 SKILL.md §3 的通用规则。本文不重复 [`mckee-setting.md §生成机制 vs 结果状态`](../mckee-setting.md#生成机制-vs-结果状态) 中"公理层 vs 机制层"的理论锚点——那段是科幻 driver 的理论原文，本文给具体名著实证。

## generative_driver 选型范畴

科幻 driver 走**两层结构**：

- **公理层**：本宇宙的基本物理 / 信息 / 社会假设（这是因的因）
- **机制层**：基于公理层推导的具体技术 / 制度 / 现象（这是 Phase 5/6 场景可直接消费的层）

公理层 + 机制层共同构成 generative_driver。**公理层缺失 → 机制层无根**（技术变成 hand-wave，读者觉得作弊）。

### platform_mode vs classic_mode

| 模式 | 特征 |
|------|------|
| **platform_mode** | 技术钩子前置（如"AI 失控""时间循环""脑机接口"）；机制层先行，公理层模糊 |
| **classic_mode** | 公理层先行（如三体的黑暗森林、相对论改写），机制层从公理推出 |

硬科幻倾向 classic_mode；爽文向科幻倾向 platform_mode。

## 名著锚点（few-shot）

### 《三体Ⅰ·地球往事》— 11 维微观结构 + 宇宙社会学公理

公理层（本部揭示）：

- **11 维微观结构**：宇宙微观尺度存在 11 维（3 宏观 + 8 微观），基本粒子内部如"活性炭"——高级文明可操控这些维度
- **光速为宇宙速度上限**：信息传播不可超光速 → 三体舰队 450 年航程的根本约束
- **宇宙社会学两公理**（本部末尾提出）：生存第一需要 + 物质总量守恒

机制层（基于公理推导）：

- **智子**：质子展开成二维后蚀刻电路再折回 → 锁死人类基础物理学（11 维结构 + 维度操控的具体应用）
- **太阳能量镜面**：太阳辐射层存在能量界面，可将电波放大 → 红岸基地通讯三体的物理基础
- **三体游戏**：宇宙规律的可视化训练场

参考产物：MUSE-canon-distill `knowledge-base/novels/三体Ⅰ-地球往事/pipeline/phase1_world.yaml`。

### 《三体Ⅱ·黑暗森林》— 黑暗森林法则（社会学公理→宇宙级博弈机制）

公理层（本部证明）：

- **宇宙社会学两公理**（Ⅰ 提出，本部证明）
- **猜疑链**：两文明之间因距离 + 物种差异，无法确认对方善意 → 形成无限递归猜疑，猜疑链不可消除
- **技术爆炸**：当前实力对比无意义——任何文明可在短时内反超

机制层（基于公理推导）：

- **黑暗森林法则**：每个文明都是黑暗森林中持枪的猎人 → 任何暴露的文明会被消灭
- **面壁计划**：因智子无法窥探内心，给四位面壁者绝对权力做秘密反制
- **水滴**：强相互作用力材料（公理 = 高级文明可武器化基本力的具体应用）

参考产物：MUSE-canon-distill `knowledge-base/novels/三体Ⅱ-黑暗森林/pipeline/phase1_world.yaml`。

### 《流浪地球》— 太阳氦闪 + 物理学硬约束

公理层：

- 太阳将在 400 年内氦闪（恒星演化物理学）
- 信息 / 物质受光速 / 重力 / 能量守恒约束
- 重元素聚变可推动行星（前沿物理学假设）

机制层：

- 12000 台地球发动机推进地球
- 分五步逃逸（刹车 → 逃逸 → 流浪Ⅰ/Ⅱ → 新太阳）
- 绕太阳变轨 + 木星引力弹弓
- 地下城（地表不宜居）

派生 world_rules：地球自转停止 → 永昼/永夜带；远日点大气冻结；危急法（撤离年龄优先）；生育权抽签

参考产物：MUSE-canon-distill `knowledge-base/novels/流浪地球/pipeline/phase1_world.yaml`。

## 三部曲 / 系列作品的公理披露纪律

逆向分析 / 写续作时严格遵循"每部只揭示该部已露的公理"——不越界预告后续：

- 三体Ⅰ：11 维结构 + 光速 + 宇宙社会学公理（提出未证）
- 三体Ⅱ：证明两公理 + 猜疑链 + 技术爆炸 + 黑暗森林
- 三体Ⅲ：维度战争 + 时间维度操控 + 二向箔

每部新揭示的公理需注明"本部首露"，不引用后续部尚未披露的设定。

## 平台样态参考（platform_mode）

观察时间：2026-05 范围。

- **起点 / 番茄常见钩子**：AI 觉醒 / 脑机接口 / 时间循环 / 异世界穿越（科幻包装） / 末世重生（科幻末世杂交） / 系统流（赛博工业风） / 星际开荒
- **盐言常见钩子**：脑洞向科幻（一刀流反转） / 末世异能 / 文明级悬念

## 高频禁区

| 禁区 | 表现 | 修正 |
|------|------|------|
| 机制层无公理根 | 直接给"时间机器 / 脑机接口"等技术，不给为什么这样可能 | 公理层先立——任何技术须有可追溯的物理 / 信息 / 社会假设 |
| 把社会后果当 driver | "阶级新分化型科幻" / "技术伦理崩坏型科幻"（SKILL.md §3 科幻行已锁） | 社会后果归 `world_rules.social`，driver 是公理 + 机制 |
| 续作越界预告 | 第一部里出现只有后续部才揭示的设定 | 严格"每部只露本部已证的公理" |
| 物理 / 信息 / 能量守恒被作弊 | 主角某次危机靠"宇宙规律突变"解决 | 守恒规则一旦立，任何破例须用公理层解释（如三体的"宇宙规则改变"也是公理级事件） |
| 用 magic 当科学 | "纳米机器人无所不能" / "量子能量解决一切" | 任何技术须给出可叙述的运作机制——不必硬科学严谨，但要内部自洽 |
