# 名著手艺模式索引

> 本文件不是仿写模板库。每条模式只抽取"承载机制"，禁止照抄名著句子。

## A. 承载点选择

### A.1 程序档案体作为整章载体（archive_chapter）
- 功能：让格式本身成为对系统性暴力的控诉，剥离一切修辞。
- 使用条件：场景涉及社会性 / 系统性暴力（连环案、战争伤亡、医疗事故）。
- 名著锚点：《2666》场景S11 — 圣特莱莎花卉区发现第一具女尸，叙事突变为"发现地点 → 衣着 → 医学伤害清单 → 调查进展"的固定档案格式，"十三岁"与"阴道和肛门都被强暴过"并置不加评论（craft_note L9-L39）。
- 可迁移规则：当受害者达到数量级 / 暴力呈现"系统性"时，切换叙事体例为档案 / 报告 / 病历；让重复格式本身说话；禁用任何叙述者情绪反应。
- 误用风险：把"档案体"当成节省篇幅的速写；档案体必须是有意的体例切换，不是偷懒。

### A.2 物质性礼物完成关系转折（material_gift_pivot）
- 功能：在高密度失败后，用一个有重量 / 有品牌 / 有送货员的物件替代"我支持你"式口头承诺。
- 使用条件：连续打击后需要一个不可逆的关系升温节点。
- 名著锚点：《2666》场景S21 — 第三本书只卖出 96 册后，布比斯送阿琴波尔迪一台 Olivetti 打字机，作家"幸福得头晕脑胀"在街上喊"出版社的人信任我啊"（craft_note L30-L36）。
- 可迁移规则：转折点的物件必须有具体品牌 / 重量 / 送达过程；不允许"她送了他一本书"式抽象表达。
- 误用风险：物件被读者识别为"作者刻意安排的象征"——避免方式：物件必须先服务实用功能（打字机能用），再带象征。

## B. 视角与信息

### B.1 跨阵营双语小人物作为战场漫游摄像机（bilingual_drifter_pov）
- 功能：让大规模战斗 / 群体事件获得一个合理的"中观"视角——不是上帝视角，不是主将视角。
- 使用条件：万人级以上的群体事件 / 会战 / 大规模仪式。
- 名著锚点：《绍宋》场景18 — 鄢陵大捷用汉奸降人洪涯（济南降人 / 前宋朝知县 / 现金军参军）作为穿越战场的摄像机，遇金人喊通古斯语遇宋军喊中原官话，求活路径就是叙事路径（craft_note L9-L15）。
- 可迁移规则：群体场景中预留一个"跨身份的小人物"槽位，他不参战只求活；他的求活路径自然成为穿越场景的视角。
- 误用风险：被改成"主将视角下的卑微随从"——必须是有独立身份焦虑（活下去 / 站对边）的角色，不是工具人。

### B.2 外视角承担屠杀（external_pov_carries_atrocity）
- 功能：让灾难场景有双 POV——内 POV 目击（绝望），外 POV 推断（恐惧），同一事件被读者经历两次。
- 使用条件：核心灾难场景，至少一个 POV 已在事件内部。
- 名著锚点：《冰与火之歌Ⅲ》场景S14 — 凯特琳在双子城内目击红色婚礼（S13），同时艾莉亚在城外看见火光、骑兵、歌声，看不见内部（craft_note L9-L57）。
- 可迁移规则：将核心灾难分配给两个空间分离的 POV；外 POV 用"帐篷数减少 / 火光颜色 / 歌声继续"等远距离信号推断；不让外 POV"突然听清楚"。
- 误用风险：外 POV 信息过满（"她虽然在城外但仿佛看见了一切"）——这是 AI 全知视角的伪装。

## C. 高潮与尺度

本节 7 pattern 严格对齐 `phase5 scene_card.climax_pattern` enum；writer 通过 scene_card 显式选择后才加载对应小节，不批量套用。

### C.1 三层揭示（layered_revelation）
- 功能：让关键真相揭示有冲击力 + 改写感，而非平铺一次性交底。
- 使用条件：场景含关键真相 / 身份反转 / 隐藏因果的揭示；scene_card.climax_pattern.primary = `layered_revelation`。
- 名著锚点：《月亮与六便士》S15 — 斯特里克兰最后画作被揭示分三层：先视觉冲击（画满整面墙的原始图景），再外行反应（医生说不出话），最后改写性事实（按遗嘱烧毁）；任一层都不附"这意味着…"式解读。
- 可迁移规则：揭示分三拍：(1) 直接感官冲击（图像 / 物件 / 声音），(2) 在场外行人物的即时反应（沉默 / 退后 / 误解），(3) 改写读者既有认知的事实陈述（一句话级别）。三拍之间不解释，不连"于是" / "这意味着" / "原来"。
- 误用风险：把三层写成"三段同等密度的描写"——三拍必须递进且体量递减；冲击层最重，事实层最短。

### C.2 不可言说的顿悟（ineffable_realization）
- 功能：让"角色突然明白了什么"具有顿悟质感，避免被翻译成可总结的命题。
- 使用条件：精神顿悟 / 美感震撼 / 信念崩塌等"无法直陈"的场景；scene_card.climax_pattern.primary = `ineffable_realization`。
- 名著锚点：《月亮与六便士》S12 — 叙述者面对斯特里克兰画作时尝试两个比喻（"像是…又像是…"），然后自我否定（"都不对"），第三个比喻不出现——"说不清"本身成为读者经历到的力量。
- 可迁移规则：让 POV 角色尝试 2 个比喻 / 命名后立刻自我否定（"不是这样"/"也不全是"），第三个不出现；用空白让读者完成顿悟，而不是替读者说出。
- 误用风险：被改成"角色想了很多比喻但其实最后说出了第三个"——一旦第三个出现就回到可言说的命题层，破坏不可言说的张力。

### C.3 被动死亡（passive_death）
- 功能：让自然死亡 / 老去场景脱离"主角主动告别"的英雄化语法，回到肉身的真实。
- 使用条件：自然衰老 / 病榻 / 平静终幕；scene_card.climax_pattern.primary = `passive_death`。
- 名著锚点：《斯通纳》S13 — 斯通纳临终时书从手中"滑落"，手指"松软"，身体"陷进"枕头；动作主语全是物件或身体部位，没有"他决定放下" / "他终于闭上眼睛"式意志陈述。
- 可迁移规则：终幕段动作的主语必须是物件（书、杯、纸）或身体部位（手指、眼睑、胸腔），不是角色本人；动词限定在非意志性词族（"松软"/"滑动"/"跌进"/"陷下"/"垂落"）；禁止"他终于…"/"他最后想着…"式临终顿悟。
- 误用风险：被理解成"死亡场景必须冷漠"——本规则不禁止情感，只禁止意志性动词；周围人物的悲伤反应可保留。

### C.4 面具硬切（mask_hard_cut）
- 功能：让硬汉 / 强势角色短暂破防后的恢复显得真实而非自然过渡，留下"那是面具"的暗示。
- 使用条件：被压抑的情绪短暂失控后角色立刻收回的场景；scene_card.climax_pattern.primary = `mask_hard_cut`。
- 名著锚点：《月亮与六便士》S14 — 斯特里克兰在某一刻流露脆弱，下一句直接切到他重新冷漠的姿态，中间无过渡句；读者感到的不是"他迅速擦掉眼泪"，而是"他主动戴上了面具"。
- 可迁移规则：破防与恢复之间用硬切（句号 + 新段，或空行）；不允许"他迅速…"/"他强忍着…"式过渡动词；恢复后的姿态描写必须带"主动选择"暗示（动作清晰、姿态精确），不是"恢复自然"。
- 误用风险：硬切被理解为"两段拼贴显得跳脱"——硬切要服务于"面具感"，前后两段必须共享同一物理位置 / 同一对手，只是姿态翻面。

### C.5 未完成动作（unfinished_action）
- 功能：让觉醒 / 反抗 / 表白等"本应释放"的高潮被堵回，传递无力比传递结果更强。
- 使用条件：角色处于觉醒边缘但被结构 / 性格 / 处境压回的高潮；scene_card.climax_pattern.primary = `unfinished_action`。
- 名著锚点：《阿Q正传》S12 — 阿Q赴刑场时觉醒台词在脑中成形（"过了二十年又是…"），但用"然而他没有说出口"立刻封闭；台词出现过，却没有完成。
- 可迁移规则：让觉醒台词或反抗动作在文本中**出现**（被读者看到），紧接着用一句封闭式短句（"然而他没有说出口" / "可是他没有抬起手" / "话已经到舌尖，又咽了回去"）立刻关闭；不允许台词被完整说出后再后悔。
- 误用风险：封闭句被写成解释（"因为他害怕…"）——封闭必须是行为陈述，不是心理解释。

### C.6 反史诗失败（anti_epic_failure）
- 功能：让主角型高潮以"主角失败 + 副角色 / 伏笔 / 世界机制完成结果"收束，避免英雄主义套路。
- 使用条件：主角企图成为英雄但失败 / 模仿前辈奇迹的高潮 / 命运需要被"非主角力量"完成的场景；scene_card.climax_pattern.primary = `anti_epic_failure`。
- 名著锚点：《指环王》末日裂隙 — 弗罗多在末日山口戴上戒指主张占有（主角失败），由咕噜抢戒指坠入岩浆完成销毁（伏笔 / 副角色欲望完成结果）；不是"弗罗多胜利但伤痛"。《冰与火之歌Ⅴ》S16 — 昆廷模仿丹妮驯龙，致命一击来自被忽视的雷哥，死亡前认知词只有"噢"（craft_note L41-L67）。
- 可迁移规则：高潮拍点上让主角的核心动作**明确失败**（不是"差一点"，是真的没做到）；结果由前文铺设的伏笔 / 副角色独立欲望 / 世界机制（重力、火、时间、人群）完成；致命转折应来自"被主角忽视的方向"。
- 误用风险：(a) 被理解为"任何高潮都该背后击杀"——本规则只针对"主角试图英雄化但失败"的场景；(b) 镜像式失败（昆廷 vs 丹妮，同样的鞭子、同样的命令词，但删除所有让前者成功的前提）是本 pattern 的可选实现手法，但镜像度建议 60-70%，太机械（每个动作一一对应）反而像戏仿。

### C.7 尺度收缩（scale_shrink）
- 功能：让宏大叙事的最终高潮收缩到一个小物件 / 一个细微动作，避免"越来越大"的史诗膨胀。
- 使用条件：宇宙 / 战争 / 文明级故事的最终拍点；scene_card.climax_pattern.primary = `scale_shrink`。
- 名著锚点：《三体Ⅲ》— 宇宙级的终结落在 5kg 生态球这一具体物件上；《指环王》— 历经魔戒大战后的真正终幕是山姆推开自家门说"我回来了"。
- 可迁移规则：宏大故事的最终拍点必须落在可触摸的小尺度上（一只手 / 一句话 / 一个物件 / 一次回家）；该小物件应在前文有过低调出场，不是空降；禁止结尾画面再做"更宏大的扩张"（黎明、星河、号角）。
- 误用风险：尺度收缩被理解为"结尾必须平淡"——平淡不是目的，**对比**才是；前文越宏大，结尾的小物件越有力，但小物件本身要承载明确情感重量。

## D. 人物刻画

### D.1 内在生存能力的失灵（inner_capacity_loss）
- 功能：把创伤的最大伤害放在"角色赖以生存的能力失灵"，而非外部事件本身。
- 使用条件：歧视 / 排斥 / 童年创伤 / 重大打击后的恢复期场景。
- 名著锚点：《你好，旧时光》场景05 — 余周周遭歧视后最大的伤害是发现自己"幻想能力失灵"——卖花姑娘的故事编不下去（craft_note L32-L42）。
- 可迁移规则：人物设计时为每个主角预留一个"内在生存能力"字段（幻想 / 写作 / 友善 / 信仰）；重大创伤场景应触及该能力的失灵。
- 误用风险：被理解为"创伤场景必须主角崩溃"——本规则的"失灵"不是崩溃，是"再也做不到曾经轻松做的事"。

### D.2 配角主体性物件（minor_character_subjectivity_object）
- 功能：让配角 / 受害者在退场场景拥有一个"独立创造物"作为主体性证据，从"被动受害"升维为"创造力的最后一闪"。
- 使用条件：女性 / 配角悲剧场景；尤其是性暴力 / 政治牺牲 / 工具人死亡。
- 名著锚点：《大唐李白》场景S20 — 月娘被强暴时最后避难所不是李白的爱情记忆，而是她自己写过的四句诗（craft_note L27-L43）。
- 可迁移规则：配角档案中应预留一个"独立创造物"字段（一首诗 / 一件手艺 / 一段记忆）；关键悲剧场景可让该物件成为她的退场承载。
- 误用风险：物件过于精雕反而像作者作秀——必须是符合配角实际能力的"创造物"，不是高级艺术品。

## E. 形态与节奏

### E.1 延迟揭示型死亡（delayed_death_revelation）
- 功能：跳过死亡现场，让死讯通过事后日常碎片渗透揭示。
- 使用条件：长篇 / 中篇有重大亲属死亡。
- 名著锚点：《你好，旧时光》场景14 — 通过公交车瞌睡、烧纸钱、午夜惊醒等碎片让读者意识到妈妈已在车祸中去世；正面车祸场景不出现（craft_note L9-L46）。
- 可迁移规则：分散在数月日常碎片中分次揭示；震撼来自读者"突然意识到"的瞬间。
- 误用风险：碎片密度不够（读者要 50 页才意识到）反而失效；建议在 5-15 页内累积到揭示阈值。
